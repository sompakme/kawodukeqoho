Is hydraulic steering the same as power steering?

All three power steering systems perform the same important function, but they execute it differently. A hydraulic system uses mechanically pumped high pressure fluid, an electro-hydraulic system uses an electric motor to pressure fluid, and an electric system uses an electric motor and sensors.
